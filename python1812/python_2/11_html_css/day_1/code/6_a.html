1995年，吴京在师傅吴彬的推荐下接拍了个人首部电影《功夫小子闯情关》，他在剧中饰演生性好武、天赋聪慧的杨学文 [13] [15] 。1998年参演电视剧《拭血问剑》，饰演西门金哥一角；同年主演的电视剧《太极宗师》在内地上映，片中吴京饰演清末民初的内家拳宗师杨昱乾 [16] 。 1999年与萧蔷、焦恩俊等演员合作，出演袁和平导演的电视剧《小李飞刀》，剧中吴京饰演阿飞一角。 [2] 2000年在电视剧《策马啸西风》中饰演孟星魂。2001年主演电视剧《江山儿女几多情》，扮演一位能文能武、沉稳威严的皇上 [17] ；同年在张鑫炎执导的电视剧《少林武王》中扮演名将后裔昙志。 [18] 2003年，吴京赴香港发展演艺事业；3月参演电视剧

《我是特种兵2》剧照 《我是特种兵2》剧照(16张) 《倩女幽魂》，饰演诸葛流云一角 [19] ；10月赴横店拍摄电视剧《南少林三十六房》。 [20] 2005参演叶伟信执导的警匪功夫片《杀破狼》，片中吴京饰演反派杀手Jet； [21] 同年接拍电视剧《武当Ⅱ》。 [22] 2006年在电影《黑拳》中饰演高岗一角 [23] 。 2007年参演电影《男儿本色》，饰演为了生存而不择手段的“天养生”，凭借这部电影的表现他获得第44届台湾电影金马奖最佳男配角提名。 [3] 2008年与萧亚轩合作拍摄MV《More More More》；12月20日，首部自导自演的电影《狼牙》在全国上映。2010年主演警匪动作电影《西风烈》，饰演性格鲁莽的警察杨晓明。 [24] 2012年10月，主演的电视剧《我是特种兵之利刃出鞘》在江苏卫视

吴京剧照 吴京剧照(13张) 播出，剧中吴京饰演90后特种兵何晨光，第一次穿上军装的吴京为了能演好这个角色，到部队前后苦练了18个月 [25] ；该剧国庆一周在江苏卫视的收视率排名第一，并在搜狐视频以近7000万的点击量排名榜首。 [26] 2013年1月8日获得搜狐时尚盛典年度时尚男明星奖 [27] ；同年自导自演军事题材的电影《战狼》 [28] 。2014年3月，担任云南卫视军旅真人秀节目《士兵突击》的第三季助梦大使。 2015年，登上央视春晚，表演武术节目《江山如画》，节目里再现了《卧虎藏龙》等电影中的功夫场面 [29] ； 战狼2剧照 战狼2剧照(2张) 4月2日，其自导自演军事题材电影《战狼》在全国上映 [30] ，该片上映首日就取得近四千万元的票房，创下国内军事题材电影首日票房纪录，上映三天累计票房已经1.37亿元 [31] ，累计票房突破5亿；6月18日，主演的功夫电影《杀破狼2》上映，片中吴京饰演卧底探员志杰 [32] ，该片首日票房拿下7200万元，第二天票房又突破4000万元，上映两天票房为1亿多元 [33] ，最终票房5.6亿元。 2016年11月，当选中国文学艺术界联合会第十届全委会委员 [34] ；同年，再次执导军事题材电影《战狼2》 [35] 。 2017年7月27日，由吴京自导自演的动作军事电影《战狼Ⅱ》上映，他在片中饰演英勇果敢的铁血硬汉冷锋；该片上映36天票房突破55亿元，不仅打破中国内地市场票房纪录、全球单一市场单片票房纪录，还成为首部进入全球TOP100票房影片榜的亚洲电影 [7] [36-37] 。他也凭借该片获得东京电影节中国电影周金鹤奖的最佳导演、最佳影片、最佳男主角 [38] ；影片获得中国-东盟电影节最佳影片奖 [39] 。 2018年2月15日，参加中央电视台春节联欢晚会，与成龙演唱歌曲《中国》 [40] ；3月17日，在第12届亚洲电影大奖上，吴京自导自演的动作军事电影《战狼Ⅱ》成为2017年最高票房亚洲电影 [41] ；11月10日，吴京凭借《战狼Ⅱ》中冷锋一角获得第34届大众电影百花奖最佳男主角奖 [42-43] 。12月29日，吴京当选中国电影家协会副主席。 [1] 2019年2月5日，主演的科幻片《流浪地球》上映 [44] 。
